=== WEBLLM IMPLEMENTATION ===

Initialization:
```javascript
import * as webllm from "@mlc-ai/web-llm";

// Check WebGPU support
const checkWebGPU = async () => {
  if (!navigator.gpu) {
    return { supported: false, message: "WebGPU not supported. Use Chrome/Edge 113+." };
  }
  return { supported: true };
};

// Initialize engine with progress tracking
const initModel = async (onProgress) => {
  const engine = await webllm.CreateMLCEngine(
    "Llama-3.1-8B-Instruct-q4f16_1-MLC",
    {
      initProgressCallback: (progress) => {
        onProgress(progress); // Update UI with download %
      }
    }
  );
  return engine;
};
```

Generation Function:
```javascript
const generateHooks = async (topic, hookType, tone) => {
  const systemPrompt = `You are a viral content strategist specializing in ${hookType}. Generate hooks that are attention-grabbing, specific, and optimized for ${tone} tone. Always deliver exactly 5-7 hooks.`;
  
  const userPrompt = `Generate 5-7 viral ${hookType} hooks about: "${topic}". 
  
Requirements:
- Each hook must be under 15 words
- Make them diverse in style
- Include ${tone} elements
- Format as a numbered list
- No explanations, just hooks`;

  const response = await engine.chat.completions.create({
    messages: [
      { role: "system", content: systemPrompt },
      { role: "user", content: userPrompt }
    ],
    temperature: 0.8,
    max_tokens: 400,
    stream: true // Enable streaming for progressive display
  });
  
  // Handle streaming response
  let fullText = "";
  for await (const chunk of response) {
    fullText += chunk.choices[0]?.delta?.content || "";
    updateUIWithPartialText(fullText); // Progressive rendering
  }
  
  return parseHooks(fullText);
};
```

Caching Strategy:
- Cache model in IndexedDB (persist across sessions)
- LocalStorage for user preferences and recent generations
- Session storage for current generation history

Error Handling:
```javascript
try {
  const result = await generateHooks(topic);
} catch (error) {
  if (error.message.includes("WebGPU")) {
    showFallbackMessage("WebGPU required");
  } else if (error.message.includes("Out of memory")) {
    showFallbackMessage("Please close other tabs and retry");
  } else {
    showGenericError("Generation failed. Please try again.");
  }
}
```
