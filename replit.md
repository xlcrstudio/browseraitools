# Browser AI Tools

## Overview
Browser AI Tools is a mobile-first, client-side web application offering a suite of AI tools that run entirely in the browser using WebLLM. The project's vision is to provide private, local, and accessible AI utilities without relying on backend API calls for core functionality. It aims to empower users with a diverse range of AI-powered generators and analyzers for creative, professional, and personal use cases.

## System Architecture
The application is built on a modern web stack:
- **Frontend**: React 18, Vite, Tailwind CSS, and Framer Motion for a dynamic and responsive user interface.
- **AI Engine**: Utilizes `@mlc-ai/web-llm` with the Qwen 2.5 1.5B model, cached in IndexedDB for offline access and performance. All AI processing occurs client-side.
- **State Management**: Leverages React's `useState` and browser `localStorage` for persistent data storage without a backend database.
- **Routing**: `wouter` for efficient client-side navigation.
- **Server**: A minimal Express server solely for serving the Vite frontend assets; no application logic or API routes are handled server-side.
- **UI/UX Design**: Emphasizes a mobile-first responsive design, featuring a distinct purple-to-blue gradient color scheme, glass-panel effects, subtle background blurs, and uses Plus Jakarta Sans for display text and Inter for body text.
- **Core Features**: The application offers a growing catalog of 53 AI tools, including various generators (e.g., Hook, CTA, Cover Letter, Startup Name, Business Idea, Keyword, Hashtag, TikTok Caption, LinkedIn Post, Resume Bullet, Blog Outline, Essay Writer, Dating Profile, Meal Planner, Travel Planner, Schema Markup, Meta Description, SEO Title, Content Brief, Excuse, Sales Email) and analyzers (e.g., Tone Converter, Interview Answer, Document Analyzer, Message Analyzer, Explain This, Content Gap Analyzer, SERP Intent Analyzer, Internal Link Suggestion). Each tool integrates unique features like history tracking, favorites, specific output formats, and structured data generation.
- **AI Sales Email Generator**: Generates 5 personalized cold outreach email variations with subject line options (3 per email), spam risk scoring, response probability predictions, "Why This Works" analysis, and "When to Use" guidance. Supports 6 objectives (book meeting, schedule demo, sell directly, introduce service, partnership, request intro), 5 tones, 3 lengths, and optional personalization merge fields. Accordion-style expandable email cards with copy-per-subject-line and copy-full-email actions. Storage key: `ai-sales-emails-v1`.
- **AI Ad Copy Generator**: Generates 5 platform-optimized ad copy variations with headlines, primary text, CTAs, and conversion scoring (0-10). Supports 5 platforms (Facebook, Instagram, Google Ads, LinkedIn, General), 4 ad formats (single, carousel, story, collection), 6 tones, 6 campaign goals, 10 preset audiences + custom, optional USP/pain point/special offer fields. Best-performing ad highlighted with star badge. Export as TXT or CSV. Accordion-style expandable cards with per-ad and bulk copy. Storage key: `ai-ad-copy-v1`.
- **AI Cold Outreach Generator**: Generates 5 personalized cold outreach message variations using different angles (Value-First, Mutual Connection, Content Appreciation, Direct/Concise, Question/Advice). Supports 4 message formats (LinkedIn Connection Request, LinkedIn Message, Email, Both), 5 discovery methods, 6 outreach purposes, 4 tones, 3 message lengths. Each message includes response probability, structure score (0-10), "Why This Works" analysis, "When to Use" guidance, and follow-up suggestions. Strongest message highlighted with star badge. Copy/save/download actions. Storage key: `ai-cold-outreach-v1`.
- **AI Landing Page Copy Generator**: Generates complete landing page copy with 6 sections: Hero (3 headline options, subheadline, CTAs), Features (5-7 with benefit descriptions), Benefits (before/after transformation), Social Proof (stats, testimonials), FAQ (objection handling), and Final CTA (closing section with risk reversal). Supports 6 page types (SaaS, E-commerce, Course, Service, App, Lead Gen), 5 tones, multi-select audiences (8 presets + custom, max 3), optional social proof fields (user count, rating, testimonial, featured-in, stats, guarantee). Each section scored for conversion effectiveness. Export as TXT or Markdown. Save/load drafts. Accordion-style expandable section cards. Storage key: `ai-landing-page-v1`.
- **AI Value Proposition Generator**: Generates 5 value proposition variations in different formats: Tagline (5-8 words), One-Liner (15-25 words), Detailed (2-3 sentences), Problem-Solution-Outcome, and Comparison. Each variation includes clarity score (0-10), analysis, and when-to-use guidance. Supports 7 industry categories. Inputs: product/service description, target audience, main benefit (all required), plus optional differentiator and industry dropdown. Best variation highlighted with star badge. Export as TXT or Markdown. Save/load drafts. Accordion-style expandable variation cards. Storage key: `ai-value-prop-v1`.
- **AI Target Audience Generator**: Generates 3 detailed buyer personas (Primary, Secondary, Tertiary) with demographics, psychographics, professional profile, goals, pain points, buying behavior, channel preferences, and messaging recommendations. Supports B2B/B2C/Both market types, 16 industries, 4 price points (Budget/Mid/Premium/Enterprise), 3 product stages. Optional advanced fields for existing customer insights and competitor focus. Each persona scored for market fit. Export as TXT or Markdown. Save/load drafts. Accordion-style expandable persona cards. Storage key: `ai-target-audience-v1`.
- **AI Elevator Pitch Generator**: Generates 5 elevator pitch variations using different angles (Problem-First 30s, Vision-First 60s, Traction-First 30s, Story 45s, Contrarian 30s). Supports 4 audience types (investors, customers, partners, general), 4 business stages, 5 desired outcomes. Each pitch includes word count, speaking time, pitch strength score (0-10), "What Works" analysis, "When to Use" guidance, and anticipated follow-up questions. Strongest pitch highlighted with star badge. Built-in practice timer (30/45/60s countdown with progress bar). Copy/save/download actions. Storage key: `ai-elevator-pitch-v1`.
- **SEO Components**: `ToolSchema.tsx` outputs `@graph` with `SoftwareApplication` + `FAQPage` schemas on every tool page. `RelatedTools.tsx` shows 5 related tools (same-category prioritized) with internal links. `ShareResultButtons.tsx` adds viral sharing (native share, X, LinkedIn, copy link).
- **Ad Integration**: Features reusable ad placeholder components (`AdBlock.tsx`) strategically placed on the homepage and within each tool page to support monetization through ad networks.
- **ATS Resume Matcher**: Jobscan-style design with 3 sequential AI calls (hard skills, soft skills + keywords, scores + suggestions). Displays color-coded found/missing skill tags, progress stepper, and sections for Hard Skills, Soft Skills, Other Keywords, Recruiter Tips, and Actionable Suggestions. Uses `SkillItem` type with `{name, found}` pairs. Storage key: `ai-ats-history-v2`.

## External Dependencies
- **@mlc-ai/web-llm**: For in-browser AI model execution.
- **React 18**: Frontend JavaScript library.
- **Vite**: Next-generation frontend tooling.
- **Tailwind CSS**: Utility-first CSS framework.
- **Framer Motion**: React animation library.
- **wouter**: Small routing library for React.
- **Express**: Minimal web server to serve static assets.
- **pdfjs-dist**: (For Document Analyzer) A library for parsing and rendering PDFs in the browser.
